# Changelog

## 0.5.1 - 2026-05-10

- Refined the OCR workspace layout to prioritize result visibility during use
- Moved the primary action closer to the result area and improved page flow
- Changed the default model to `gpt-5.2`
- Added provider-aware compatibility settings, including Anthropic Claude native API support
- Split the project documentation into separate English and Simplified Chinese README files

## 0.5.0 - 2026-05-10

- Renamed the public project identity to `Browser Image OCR`
- Upgraded the extension page UI to a more product-like layout
- Added visible version and upgrade guidance inside the OCR page
- Improved repository docs for open-source publication

## 0.4.0

- Added support for custom OpenAI-compatible endpoints
- Added configurable API mode and image transport mode
- Added OCR result post-processing for links and GitHub repositories
