网页图片右键 OCR，通过 OpenAI-compatible 与 provider 兼容视觉接口高精度提取文字、链接和截图内容。

## Full Description (English)

Browser Image OCR is a lightweight Edge/Chrome extension for extracting text from images directly in the browser.

After installation, you can right-click almost any image on a web page and send it to a compatible vision model endpoint for OCR. It is designed for screenshots, posters, social cards, link previews, technical diagrams, and mixed Chinese-English content.

Key capabilities:

- Right-click any web image to start OCR instantly
- Works with OpenAI-compatible endpoints and relay services
- Supports provider-aware compatibility settings
- Supports Anthropic Claude native API mode
- Supports both `Chat Completions` and `Responses` style APIs
- Supports direct image URL mode and Base64 mode
- Helps detect and extract links from OCR output
- Keeps API keys only in local browser extension storage

Typical use cases:

- Extract text from screenshots or posters
- Capture links from social cards or shared images
- Read mixed Chinese-English UI screenshots
- OCR technical content, repository cards, and developer screenshots

Notes:

- This extension requires you to provide your own compatible API endpoint and API key
- Unpacked local installations do not auto-update
- Automatic updates typically require publishing through an extension store

## Full Description (Chinese)

Browser Image OCR 是一个轻量的 Edge/Chrome 浏览器扩展，用于直接在浏览器里识别网页图片中的文字。

安装后，你可以在网页图片上点击右键，把图片发送到兼容的视觉模型接口做 OCR。它适合识别截图、海报、分享卡片、链接预览、技术图示，以及中英文混排内容。

核心能力：

- 右键任意网页图片，快速发起 OCR
- 支持自定义 OpenAI-compatible 接口和中转服务
- 支持按 provider 区分兼容方式
- 支持 Anthropic Claude 原生接口模式
- 同时支持 `Chat Completions` 和 `Responses` 两种接口风格
- 支持图片 URL 直传和 Base64 两种图片传输方式
- 可从 OCR 结果中辅助提取链接
- API Key 只保存在本地浏览器扩展存储中

典型使用场景：

- 从截图、海报中提取文字
- 从分享卡片或社交图片中抓取链接
- 识别中英文混排的界面截图
- 识别开发者截图、仓库卡片和技术内容

注意事项：

- 该扩展需要你自行提供兼容的 API 接口和 API Key
- 通过“加载解压缩的扩展”安装时，不支持自动升级
- 如果要自动升级，通常需要通过扩展商店发布并安装
